Preparation

What do I need?

  • A Google Account
  • A Google API Project with Google Maps API v3 service turned on. (Google API Console)
  • A place to put your code so you can view it in a browser
  • A decent web code editor. Not word processing software like MS Word









































Lesson 1

Let's make the page to put the map on.

  1. Create the /js directory

    In your web site location's document root, create a /js directory. This will eventually hold your Javascript code to make the map.

  2. Create the index.html page

    In your web site location's document root, create an index.html page that looks like lesson1.html

  3. Insert your Google API Key

    In your index.html page code, replace 'YOUR_GOOGLE_API_KEY_HERE' with the API Key found in your Google API project.
